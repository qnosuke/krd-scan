このアプリについて

オムロン体組成計 カラダスキャン KRD-203 の測定値をカメラで読み取り、 記録・CSV書き出しするための個人開発アプリです。

プライバシー

  • 測定データを開発者は一切保存・収集していません。 データはすべてお使いの端末の中（ブラウザの保存領域）にのみ保存されます。 開発者側にはサーバーそのものが存在せず、データの送信先がありません。
  • カメラ映像も端末内でのみ処理され、外部に送信されません。
  • アクセス解析・トラッキング・Cookie・広告は一切ありません。 開発者は利用状況もトラフィックも見ていません（見る仕組み自体を入れていません）。
  • 本アプリは静的ファイルとして GitHub Pages から配信されています。 配信インフラとしてのアクセス記録の扱いは GitHubのプライバシーステートメント に従います（開発者に個別のアクセス情報は提供されません）。
  • ソースコードは GitHubですべて公開 されており、上記を誰でも検証できます。

データの保存場所と注意

  • 記録はブラウザ（Safari等）のサイトデータとして保存されるため、 サイトデータを削除すると記録も消えます。
  • 大切な記録は「履歴」タブから定期的にCSVで書き出して保管してください。
