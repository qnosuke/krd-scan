# カラダスキャン記録（karadascan）

オムロン体組成計 **カラダスキャン KRD-203-W** の測定値を、iPhoneのカメラで
液晶から読み取って記録し、CSVで書き出せるPWA（Webアプリ）です。

KRD-203 にはデータ保存・スマホ連携機能がなく、測定結果（7項目）が約1秒ごとに
表示で切り替わるだけで消えてしまうため、その表示をカメラで自動読み取りします。

## 使い方

1. iPhoneのSafariでアプリのURLを開き、共有メニューから「ホーム画面に追加」
2. 「計測」タブで **読み取り開始** を押してから体組成計に乗る
3. スマホのカメラを液晶の数字にかざすと、切り替わる7項目
   （体重・体脂肪率・内臓脂肪レベル・骨格筋率・体年齢・基礎代謝・BMI）を自動で読み取ります
4. 確認画面で誤読を修正して保存
5. 「履歴」タブからCSVを書き出し（共有シートでファイル保存・AirDrop・メール等）

読み取りがうまくいかない場合は、計測中の液晶を動画で撮っておき、
「取り込み」タブから動画を読み込ませることもできます。

## プライバシー

- **測定データを開発者は一切保存・収集していません。** データはすべて端末の中
  （ブラウザのIndexedDB）にのみ保存されます。開発者側にはサーバーそのものが存在せず、
  データの送信先がありません
- カメラ映像も端末内で処理され、どこにも送信されません
- **アクセス解析・トラッキング・Cookie・広告は一切ありません。** 開発者は利用状況も
  トラフィックも見ていません（見る仕組み自体を入れていません）
- 本アプリは静的ファイルとして GitHub Pages から配信されており、配信インフラとしての
  アクセス記録の扱いは [GitHubのプライバシーステートメント](https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement)
  に従います（開発者に個別のアクセス情報は提供されません）
- ソースコードは本リポジトリですべて公開されており、上記は誰でも検証できます

### データの保存場所と注意

記録はブラウザのサイトデータとして保存されるため、サイトデータを削除すると記録も
消えます。大切な記録は「履歴」タブから定期的にCSVで書き出して保管してください。

## 免責

読み取り結果には誤りが含まれる可能性があります。保存前に確認画面でご確認ください。
本アプリはオムロン ヘルスケア株式会社とは無関係の非公式ツールです。

## 開発

```sh
npm install
npm run dev      # 開発サーバー
npm test         # ユニットテスト（認識エンジン等）
npm run build    # 本番ビルド（GitHub Pages用 base=/karadascan/）
python3 scripts/gen-icons.py  # アイコン再生成
```

- 7セグ認識エンジン: [src/sevenseg.js](src/sevenseg.js)（依存ライブラリなし）
- 項目割り当てロジック: [src/session.js](src/session.js)
- 機器の表示仕様: [docs/spec.md](docs/spec.md)
